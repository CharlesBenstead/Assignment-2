Implementation styles

Inline

You can find more on inline here as explained on that page, the inline style implementation is done by adding the style attribute to a chosen HTML element. This adds a specific style to just the element chosen. This is especially useful for when a user wants a certain element to have a specific style, different to the rest of the page, or if there is not a .css file available.
